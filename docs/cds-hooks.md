# CDS Hooks

本專案保留 CDS Hooks 範例能力，核心程式在 `src/cds-hooks.js`。

## 相關檔案

- `src/cds-hooks.js`
- `examples/cds-hooks-server.js`

## 啟動方式

```bash
npm run cds-hooks-server
```
