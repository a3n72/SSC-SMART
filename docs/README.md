# 文件總覽

本專案的正式文件集中於此，`examples/` 只保留可直接執行的頁面、腳本與測試資料。

## 主要文件

- `smart-launch.md`: SMART on FHIR 啟動、私人 launcher 相容性、`client_id` / `client_secret` 設定方式
- `project-structure.md`: 專案結構、核心模組分工與維護建議
- `testing-and-data.md`: 第二階段測試準備、FHIR 測試資料與執行檢查清單
- `cds-hooks.md`: CDS Hooks 範例與服務端使用方式
