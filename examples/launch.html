若由私人 Launcher 或 EHR 直接導向本頁，系統會自動帶入對應的 `iss` / `launch`
可填 FHIR Server 或私人 Launcher 回傳的 `iss`，未填時會優先使用 `app-config.js` 的預設值
若可讀取 SMART `.well-known` 設定會自動帶入，否則依 ISS 網域推算
授權碼交換 Access Token 的端點，若可讀取 SMART 配置會自動帶入
請填入測試單位提供或已註冊的真實 `client_id`
僅限測試或受控環境使用。正式前端不建議直接保存 secret。
